### 회귀분석

- 회귀 분석은 입력 변수 X에 대 연속형 출력 변수 Y를 예측한다.
- 오차(잔차)를 기반으로 분석 결과를 평가한다.
- MAE는 오차를 절대값으로 변환해 평균을 낸 것 
- MSE는 오차를 제곱해서 평균을 낸 것. 
- MSE에 루트 적용 -> RMSE 
- MSE에 로그 적용 -> MSLE
- MSLE에 루트 적용 -> RMSLE

예측값 = 100, 실제값 = 90인 경우 RMSE = 10, RMSLE = 0.1053
예측값 = 10000, 실제값 = 9000인 경우 RMSE = 1000, RMSLE = 0.1053

R² : 분산 기반으로 예측 성능을 평가 
- 예측값의 분산 / 실제값의 분산
